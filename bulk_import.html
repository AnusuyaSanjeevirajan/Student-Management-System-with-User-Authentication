📤 Bulk Import Students

Use this tool to quickly add many students at once from a CSV file. Make student data entry fast, clean, and visual.

📋 CSV Format Requirements

Your CSV file should have these columns in this order:

name,email,phone,roll_number,course,enrollment_date,address
  • Example row: John Doe,john@example.com,9876543210,STU003,Computer Science,2024-01-15,123 Main St
  • Example row: Jane Smith,jane@example.com,9876543211,STU004,Information Technology,2024-01-16,456 Oak Ave
  • Only name, email, and roll_number are required.
  • Other fields can be empty if you do not have the data yet.
